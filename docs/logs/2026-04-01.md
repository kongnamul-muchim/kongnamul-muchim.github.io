## 📂 3. Unity Hierarchy 스타일 메뉴

### 폴더 트리 구조
```
📂 Projects
├── ▼ 🎮 Unity (4)
│   ├─ 🎯 DefenceGame
│   ├─ ⚔️ GreedDungeon
│   ├─ 🗺️ Pathfinder
│   └─ 🎯 ShotUp
├── ▶ 🌐 Web (1)
├── ▶ 📦 Package (1)
├── ▶ 🖥️ Server (1)
└── ▶ 📄 Docs (1)
```

### 기능
- 폴더 클릭 → 확장/접기 토글
- 다른 폴더 열면 자동 닫힘
- 프로젝트 클릭 → 페이지 이동
- 화살표 아이콘 회전 (▶ → ▼)

### 기술적 문제 해결
1. **클릭 안됨**: Astro script module 문제 → `DOMContentLoaded` 이벤트 래핑
2. **줄바꿈 안됨**: `white-space: nowrap` 제거 → `word-break: break-word` 적용
3. **색상 불일치**: 파란색 완전 제거 → 메탈릭 그레이 통일
