## 프로젝트 개요

Next.js 기반의 커뮤니티 게시판 시스템입니다.

### 주요 기능

- 게시글 CRUD
- 사용자 인터랙션
- 커뮤니티 기능
- 반응형 디자인

### 기술 스택

- Next.js App Router
- TypeScript
- Tailwind CSS
- Prisma ORM
